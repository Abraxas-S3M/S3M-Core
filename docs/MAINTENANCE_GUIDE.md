# S3M Maintenance Guide

Classification: UNCLASSIFIED - FOUO

## Log Rotation
- Rotate threat logs at 100 MB
- Rotate audit logs at 100 MB
- Rotate decision logs at 100 MB
- Archive replay files weekly

## Model Updates
1. Stage new weights on trusted host
2. Verify checksums
3. Transfer via approved media
4. Update registry and restart services

## Security Audits
- Weekly: compliance check
- Monthly: vulnerability scan

## Performance Baseline
- Quarterly: run benchmark suite
- Compare against baseline report

## Data Cleanup
- Remove expired threat events
- Purge stale synthetic datasets
- Archive old replay files

## Backup Schedule
- Daily: config backup
- Weekly: audit archive
- Monthly: full mission data snapshot
